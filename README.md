# C++ Programming

A beginner-friendly collection of basic C++ programs. Each example is designed for learning, practice, and easy reference.

---

## About this Repository
- Covers core C++ topics with simple programs.
- Each program file is intended to include:
  - problem description
  - input/output examples
  - sample output
  - source code

---

## How to Use
1. Open the desired `.cpp` file from the `codes/` folder.
2. Compile it with `g++`.
3. Run the generated executable.

Example:

```bash
g++ codes/add-two-numbers.cpp -o add-two-numbers
./add-two-numbers
```
